font-size : 글꼴의 크기 지정 속성

        * 고정 크기 단위
        px : 웹 개발의 표준 단위 픽셀 - 화면의 해상도
        pt : 포인트(1pt = 0.35mm)

        * 가변 크기 단위
        % : 상위 요소 크기에 대한 백분율
        em : 상위 요소의 글꼴(font)크기에 대한 비율(1em == 100%, 
                                                    1.2em == 120% )
  • 픽셀(px)
  • 포인트(pt)
  • 퍼센트(em)

font-weight : 글꼴의 굵기(두께)를 지정하는 속성

        bold : 굵게(700)
        normal : 보통(기본값, 400)

        bolder : 부모 요소보다 굵게
        lighter : 부모 요소보다 얇게

        number(100~900, 100단위) :  가중치
            - 숫자가 클수록 굵기가 굵은
            - font에서 지원하는 가중치에 따라 표현되는 굵기가 다름

            -> normal, bold만 지원하는 폰트일 경우
            100~500 : normal
            600~900 : bold

font-family : 글꼴을 지정하는 속성

        * 폰트명(궁서) : 글자 크기에 따라 변하는 폭(가변)
          폰트명+체(궁서체) : 글자 크기와 관계없이 일정한 폭(고정)

        *  font-family: 폰트명을 여러 개 작성 가능하고
           폰트가 존재하지 않는 경우, 브라우저의 기본 폰트가 적용된다.
