레이아웃(layout)

    사전적 의미 : 배치, 정리
    기술적 의미 : 구성 요소(html요소)를 제한된 공간(html body)에 효율적으로 배치하는 것

화면 배치 방법(형식) : display속성

    요소가 화면에 어떻게 보여질지 형식을 지정하는 속성
    * block 
        - 화면을 수직 분할(행 나눔)
        - width, height 사용 가능     

    * inline 
        - 화면을 수평 분할(한 행의 열 나눔)     
        - width, height 사용 불가능!!

    * inline-block 
        - inline의 수평 분할 + block의 크기 조절 

    * none 
        - 화면에 요소가 표시되지 않으나 존재하는 상태

    * flex 
        - 요소에 정렬되는 방향, 간격을 유연하게 처리하는 방식
        - 브라우저가 해석하여 처리

화면(영역)분할

상하 2분할

    상하 2분할을 위한 준비물
    - 감싸고 있는 영역 요소
    - 내부를 분할할 요소
    - display: block
    - 크기 조절을 위한 크기 지정 단위: px, %(가변)

상하 3분할

    1) 감싸는 전체 영역
        - id: container-2
        - 높이: 440px;
        - 너비: 300px;

    2) 내부 분할 영역 비율 
        - 15 : 35 : 50

좌우 2분할

    좌우 2분할을 위한 준비물
    - 감싸고 있는 영역 요소
    - 내부를 분할할 요소
    - display: inline-block
    - 크기 조절을 위한 크기 지정 단위: px, %(가변)
