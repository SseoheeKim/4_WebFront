요소의 영역(여백)관련 속성

    모든 html 요소는 총 4가지의 영역으로 구성

    1. content
        - 요소의 내용 작성
        - 시작/종료 태그 사이에 작성되는 내용이 출력되는 영역

    2. padding
        - content와 border사이의 영역

    3. border
        - 요소의 테두리가 지정되는 영역
        - content보다 바깥 쪽에서 감싸고 있는 테두리

    4. margin
        - 다른 요소와의 간격을 지정하는 영역

box-sizing

    요소의 전체 크기가 가로/세로 300px인 정사각형

    * box-sizing 속성
    - width, height 속성이 적용되는 범위를 지정하는 속성
    1) content-box(기본값) : width, height 적용 범위를 content로 한정

    2) border-box
        width, height 적용 범위가 
        border + padding + content 크기 합산과 동일하도록 계산해서 
        content의 크기를 자동으로 지정 

    사용 예시)
    * {
        margin: 0px;
        padding: 0px;
        box-sizing: border-box;
    }
