CSS(Cascading Style Sheets)

    HTML과 같은 마크업 언어가 실제로 화면에 표시되는 방법(모양, 색상, 크기 등)을 기술하는 언어
    (HTML코드가 화면에 어떤 스타일로 보여질지 지정하는 언어)
    
    W3C(웹 표준 지정 기관)에서 CSS를 표준으로 지정

CSS Selector(CSS선택자)

    선택자란?
    현재 HTML 문서 내에서 특정한 요소를 선택하는 문자 또는 문자열

태그 선택자

    현재 HTML 문서 내에 같은 태그를 선택하는 선택자
    -> 원치 않는 태그들도 모두 다 선택

    [작성법]
    태그명{ css코드; }
strong태그입니다
span태그입니다

id 선택자(#id 속성값)

    HTML 문서 내에서 id속성값이 일치하는 요소를 선택하는 선택자
        
    (중요!!!!) id 속성값 == 식별자
    id 속성 값은 페이지 내에서 '유일한' 값이어야만 한다!
    -> 하나의 HTML 문서 내에서 중복 되는 id가 존재해선 안된다
        
    [작성법]
    #id속성값{ CSS코드; }

    # == id

Class 선택자(.class 속성값)

    HTML 문서 내에서 같은 class 속성 값을 가진 요소를 '모두' 선택하는 선택자
    id속성값은 중복X, class는 중복O

    [작성법]
    .class 속성값{ CSS코드; }

    . == class
