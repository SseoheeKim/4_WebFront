스크립트 언어

        기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도의 언어

        별도로 컴파일을 진행하지 않고,
        인터프리터(interpreter)를 이용하여 소스코드를 한 줄씩 읽어 바로 실행

        -> 컴파일에 소요되는 시간은 따로 필요없지만
        한줄 한줄 읽기 때문에 프로그램 실행 시간이 느림

Javacript란?

        자바를 참고한 언어로 객체지향 지원하지만 전혀 다른 것으로
        웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체 지향 프로그래밍언어

        ECMA Script 표준을 따르는 대표적인 웹 기술

자바스크립트 작성 방법

  • 브라우저 콘솔에 직접 작성
  • 태그 내 직접 JS 코드를 작성(inline방식)
  • html 내부에 script태그를 이용하여 작성(internal)
  • html 외부에 파일(.js)을 이용해 작성(external)
