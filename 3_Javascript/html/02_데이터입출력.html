innerHTML

        자바스크립트에서 요소 전체를 읽어들이거나 변경하는 속성
        
        - 내용을 읽어올 때 태그 + 속성="속성값" + 내용(content)을 모두 포함
        - 내용을 변경할 때 태그는 HTML요소로 해석

안녕하세요?

innerHTML 응용

예) input 태그 입력 시 마다 추가될 때마다 하나씩 추가되도록 할 때

innerText

        자바스크립트에서 요소에 작성된 내용(content)만을 읽어들이거나 변경하는 속성

        - 내용을 읽어올 때 요소의 태그와 속성은 무시
        - 내용을 변경할 때 대입되는 태그는 모두 문자열로 해석 (HTML 태그로 해석 X)

alert("내용")

        - alert(알리다, 경보, 경고)
        - 브라우저에 대화상자(알림창)을 띄우는 함수

confirm("내용")

        질문에 대한 "예" / "아니오" 결과를 얻고자 할 때 사용하는 대화상자
        - 예/확인     : true반환
        - 아니오/취소 : false반환

prompt("내용")

        텍스트를 작성할 수 있는 대화상자
        - 확인 : 입력한 값을 반환
        - 취소 : null 반환
