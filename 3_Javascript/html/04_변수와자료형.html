변수의 선언 방식

        Javascript는 변수 선언 시 자료형이 아닌 var, let, const, (미작성) 중 선택한다.

        var    : 변수, 변수명 중복 가능 O(기존의 변수에 덮어쓰기), 함수 레벨 scope
        let    : 변수, 변수명 중복 불가 X, 블록{ } 레벨 scope
        const  : 상수, 변수명 중복 불가 X, 블록{ } 레벨 scope
        미작성 : var와 동일


        1순위 : const
                (JS는 HTML상에서 특정 요소를 선택해 지속적으로 추적하는 경우가 많아
                상수형 변수에 저장해 고정적으로 사용)
        2순위 : let
        3순위 : var
        비추천 : 미작성

Javascript의 자료형

        JS는 변수 선언 시 별도의 자료형을 지정하지 않는다.
        -> 변수에 대입되는 값(리터럴)에 의해서 자료형이 결정된다.

        - String (문자열 /"abc", 'abc')
        - number (숫자 / 100(정수), -3.14(실수))
        - boolean (논리값 / true, false)
        - object (객체 / 배열(Array), JS객체 {K:V, K:V} )
        - function (함수)
        - undefined (자료형이 정의되지 않은 변수) 
                ㄴ 값이 대입되지 않은 변수


        * null은 참조하는 게 없는 값을 나타내는 리터럴(자료형X)
