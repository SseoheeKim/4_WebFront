배열(Array)

        배열: 변수를 묶음으로 다루는 것(변수가 연속적으로 나열됨)
        JS  변수 특징 : 값이 저장되기 전 까지 자료형이 지정되지 않음.

        * JS 배열 특징
        1. 자료형 제한 X
        2. 길이 제한 X
        == Java의 Collection List와 비슷
    
        1) const arr1 = new Array(); // 0칸짜리 배열 생성
        
        2) const arr2 = new Array(3); // 3칸짜리 배열 생성
        
        3) const arr3 = []; // 0칸짜리 배열 생성
        
        4) const arr4 = ['사과', '딸기', '바나나']; // 0칸짜리 배열 생성
                        // 3칸짜리 초기화된 배열 생성
