글자 관련 태그

b 태그 : 단순히 글자를 굵게 출력
i 태그 : 단순히 글자를 기울이게 하여 출력

strong 태그 : 중요성, 심각함을 강조
em 태그 : 문장 일부분의 중요성을 강조

abc → 태그는 중복 적용이 가능

*b/i과 strong/em 사용의 차이 출력 형식은 동일하지만 1)태그 자체 의미의 차이, 2)웹 접근성에서 차이가 있다.

mark 태그 : 형광펜 효과
u 태그 : underLine(밑줄)
s 태그 : 취소선 → 잘못된 것을 표시
del 태그 : 삭제선 → 없어진 텍스트를 표시
small 태그 : 작은 글자 표시
sup태그(윗 첨자) ex) x2
sub태그(아래 첨자) ex) log10

abbr(abbreviation)태그(약어) : 마우스 오버 시 툴팁 출력
ex) HTTP

글자 관련 태그 응용

태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은글자를 사용할 수도 있고, 기울이거나, 취소선을 넣는 등
다양하게 사용할 수 있다.
