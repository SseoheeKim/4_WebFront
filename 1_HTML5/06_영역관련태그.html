영역 관련 태그

    
블럭(Block) 형식

    - 공간을 수직 분할
      : 수직인 화면에 가로선을 그어 여러 행으로 분할
    
    - 크기를 지정하는 css속성인 width, height 사용 가능

    ex) p/pre(문단 나누기), 
        h1~h6(제목 영역 나누기),
        hr(수평선으로 화면 나누기), 
        div(영역 나누기) 등

    
인라인(InLine) 형식
                 
    - 공간을 수평 분할
    : 수평인 화면에 세로선으로 그어 여러 열(컬럼)로 분할
  
    - 크기를 지정하는 css속성인 width, height 사용 불가!! 
    
    ex) strong/b(글자를 굵게), 
        em/i(글자 기울리기),
        mark(글자 노란배경),
        span(한 줄 내에서 열을 나누는 것) 등

inline(수평분할)

첫 번째 영역두 번째 영역 세 번째 영역 
        inline형식의 태그는 한줄 내에서 작성된 내용만큼 
        칸(열) 차지하기 때문에 다음에 작성된 내용이 줄을 바꾸지 않고 붙어서 출력
        
        * inline 형식의 태그 이용시 유의사항
        html문서에 엔터를 입력해서 작성하는 경우
        출력되는 화면에서 한칸 띄어쓰기 된 형태로 출력된다!
        -> 붙여서 출력하고자하면 엔터.띄어쓰기 없이 태그 작성
