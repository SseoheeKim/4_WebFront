text관련 input태그

    ** text관련 타입의 공통 속성 **
    - size : 입력 상자의 크기 지정
    - maxlength : 입력가능한 문자열의 최대 길이

    - placeholder : 입력 상자에 작성될 내용 설명
        (text 뿐만 아니라 다양한 number type에서도 사용 가능)

type="text"

    - 한 줄 짜리 문자열을 입력할 수 있는 입력 상자   
    - input 태그의 기본값(type작성하지 않아도 type="text")
    
    ** 모든 input태그의 형식은 inline

label 태그

    - input태그 제목으로 사용되는 방법

    - 사용방법
    1) input태그에 id속성 값을 작성
    2) 원하는 위치에 label태그를 작성한 후 for 속성을 작성
        이때 for 속성의 값은 1)에서 작성한 id 속성값과 같아야 한다.


form 태그

    - 내부에 작성된 input태그의 값을 서버로 전달
        -> 이 때 전달되는 값의 형식은 K:V형태
        key ==  input태그의 name 속성 값
        value == input태그에 작성된 값 또는 value속성 값

    - form태그의 속성
        1) ** action : 내부에 작성된 input값을 어디로 전달할지 지정하는 속성
                    -> 주소를 작성
        
        2) method(방식) : 데이터를 서버로 전달할 때의 방식을 지정하는 속성

        3) name : form태그 이름을 지정(JS사용)

        4) target : action에 지정된 주소를 새 탭/현재 탭 중 어디서 열지 지정
