하이퍼 링크(하이퍼 텍스트) 관련 태그

    - Hyperlink(Hypertext) : 다른 문서로 즉시 접근/이동하는 텍스트

    - HTTP(HyperText Transfer Protocol) : 하이퍼텍스트로 이루어진 문서(HTML)를 전송하는 통신 규약

a(anchor)태그

    - a태그는 현재 HTML문서와 다른 HTML문서를 연결해서 이동할 수 있게 하는 기능을 제공
      (페이지 이동) 
    
    - a태그는 inline형식 

    ** 하이퍼링크는 웹 문서가 다른 문서와 구분될 수 있는 핵심 기능
        1) 다른 페이지로 이동 가능
        2) 현재 페이지 내에서 이동 가능

href속성 : 참조할 페이지(이동할 페이지)의 주소를 작성하는 속성

target속성 : 연결된 문서를 어디에 열 것인지 지정하는 속성
